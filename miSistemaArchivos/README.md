## Compilación y uso

Empaquetamos el programa en un jar con: 
```python
mvn package
```

Luego ejecutamos en ./target con un archivo de disco vacio archivo.fs:

```python
java -jar miSistemaArchivos-1.0.0.jar disco.fs
```
